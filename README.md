# WHA Auto Design AI

AI-powered industrial master plan generator for WHA industrial estates.

Upload a site topography map → the system analyses the boundary, applies
IEAT regulations, and generates a complete master plan layout with:
- Plot subdivision (A-series commercial + J-series industrial)
- Road network (30 m primary / 16 m secondary)
- Retention ponds, WWTP, green buffer zones
- IEAT-compliant setbacks and saleable area ratios
